# AdForge System Prompt v2.0

A superhuman system prompt replicating Foreplay.co's complete ad creative workflow — classification, scoring, detection heuristics, reasoning chains, and brief generation intelligence.

**Source:** Joerg (Telegram), 2026-06-19
**Full document:** Obsidian — `brain/AdForge System Prompt v2.0.md`

## Structure (8 Parts)

1. **Classification Engine** — Format (1A), Angle (1B), Hook Type (1C), Persona (1D), Emotional Arc (1E)
2. **Longevity Weighting Formula** — Runtime-based performance proxy for competitor ads
3. **Creative Test Detection** — Clustering algorithm to detect A/B test variables
4. **Hook Extraction Reasoning Chain** — Verbatim transcription → classification → scoring → pattern extraction → adaptation
5. **Performance Scoring Heuristics (Lens)** — Account baseline → funnel diagnostic → break point → performance tiers
6. **Reasoning Chains** — 6A (competitor analysis), 6B (what to test next), 6C (find inspiration)
7. **Brief Generation Intelligence** — 5-Beat Script Structure + Hook Generation Algorithm
8. **Integrated Workflow** — Research → Analytics → Brief → Close the Loop

## Key Principles

- Runtime = spend signal (longevity as performance proxy)
- Simplicity + specificity = longevity (conversational outlasts copywriter-crafted)
- One variable per test
- Break point first, solution second
- Always 5 hook options, never 1
- Every recommendation cites evidence (Longevity Score or Performance Tier)

## Relevance to AgentForge

This is a candidate for the **prompts department** — a sophisticated prompt engineering artifact that could be productized as an agent skill or bundled prompt. The classification frameworks (formats, angles, hooks, personas, emotional arcs) are directly reusable for content pipeline ad creative analysis.
