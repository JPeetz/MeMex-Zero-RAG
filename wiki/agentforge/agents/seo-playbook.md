# SEO Quality Gate — Playbook

_Lessons from audit runs._

## Common Failure Patterns
- Missing meta description
- Flesch score too low (<30)
- No hyperlinks in first 3 paragraphs
- Title too long (>60 chars mobile truncation)

## Fix Patterns That Work
- FLAG at 65 → add meta description + hyperlinks → PASS at 79
- Low Flesch → add 3-4 plain-language interludes → +5-8 points

## GEO Structural Requirements
- Definition block: 45-55 words
- FAQ block: 4-6 questions
- Quotable summary: 60-80 words
- Featured image: required
