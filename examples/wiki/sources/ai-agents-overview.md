# AI Agents Overview (2026)

A survey of the AI agent landscape as of April 2026, covering definitions, components, major frameworks, and challenges.

## Definition

AI agents are autonomous systems that perceive their environment, make decisions, and take actions to achieve goals. Unlike chatbots, they use tools, maintain state, and execute multi-step workflows. [Source: raw/ai-agents-overview.md]

## Core Components

1. **LLM Core**: Reasoning engine (GPT-4, Claude, Gemini)
2. **Tool Use**: APIs, code execution, file access
3. **Memory**: Context window + external storage
4. **Planning**: Task decomposition

[Source: raw/ai-agents-overview.md]

## Key Challenges

- **Hallucination**: Confident wrong actions
- **Context limits**: Long workflows overflow
- **Cost**: Token consumption adds up
- **Reliability**: Inconsistent behavior

[Source: raw/ai-agents-overview.md]

- [[human-in-the-loop]]: Agents propose, humans approve
- Specialized > general-purpose agents
- Agent-to-agent collaboration
